# ProjectE — MC 26.1.2 port

Unofficial port of ProjectE for **Minecraft 26.1.2** / **NeoForge**.  
Maintained by **Yaskulsky**.

Transmutation tables, collectors, condensers, rings, EMC — full ProjectE / EE2-style gameplay.
