# Discord server setup — Yaskulsky / Equivox 26.1 port

Copy-paste into your server. Replace `[CURSEFORGE_LINK]` when you have it.

---

## Server settings

| Field | Value |
|-------|-------|
| **Server name** | `Equivox 26.1 Port` or `Yaskulsky Mods` |
| **Description** | Unofficial Equivox port for MC 26.1.2 / NeoForge. Not affiliated with official Equivox. |

**Invite:** Server Settings → Invites → never expire → paste link in README + CurseForge.

## Channel structure

Create categories and channels (right-click server → Create Category / Create Channel):

```
📢 INFO
  #rules
  #announcements
  #links

💬 COMMUNITY
  #general
  #support
  #bug-reports
  #suggestions
```

Optional later: `#dev-updates`, `#pack-testing`, voice lounge.

## #rules (pin this)

```
📌 Server rules — Equivox 26.1 port (unofficial)

1. This is an UNOFFICIAL community server for Yaskulsky's Equivox port (MC 26.1.2 / NeoForge). We are NOT the official Equivox team.

2. Be respectful. No harassment, spam, or NSFW.

3. English and Polish are fine.

4. Bug reports: use GitHub (required for fixes):
   https://github.com/Yaskulsky/projecte-26-port/issues
   Include: MC version, NeoForge version, mod list, log/crash report.

5. Do not share fake builds, leaked files, or claim this is an official release.

6. The mod is free (MIT). Support is optional: https://buymeacoffee.com/yaskulsky

7. Back up your world before updating the mod.
```
